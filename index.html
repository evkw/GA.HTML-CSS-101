SO, WHATS WEB DEVELOPMENT?

What's Web Development?
  • The creation and management of both websites and web-based applications
  • Made up of the Front End and Back End
  • Mainly involves editing text files and making sure they are in the right places

web production workflow

web production workflow
  • Take the design and turn it into code and assets.
  • Uses HTML/CSS to create the structure for a page and add styles.
  • Add interactions with Javascript.
  • Other responsibilities: accessibility, performance, cross-browser, offline first and cross-device functionality.
what does front end even mean?

“A mix of programming and layout that powers the visuals and interactions of the web.”

in summary...
  • The Front End is what the user sees
  • It powers the visuals and interactions of the web
  • Made up of HTML, CSS and Javascript
  • Is constantly evolving. Trends and styles come in and out of fashion.

A brief look at html + css

html
  • Structure of the cake
  • Made up of building blocks (cake tiers in this analogy)
css
  • Style of the cake!
  • Allows us as developers/designers to bring our own unique flavours to our work
  • Some of the most satisfying, but most stressful parts of front end development
javascript
  • Adds interactions to the cake
  • Where the majority of programming concepts come into practice
  • Allows us to tackle really large problems

HTML Layout
ACTIVITY: CREATING YOUR FIRST HTML FILE
  1. Create a folder on your desktop and title it “Coding 101”
  2. Open up the folder you just made in your text editor
  3. Save your file as “index.html”
  4. Type out the HTML basic layout
  5. Open up the HTML file in Google Chrome
  6. Add more content tags: h1-h6, p, ul/li, button or anchor
  7. Keep your editor open

BASIC LAYOUT OF AN EXTERNAL CSS FILE

CSS: Cascading style sheet
CSS: Example
ACTIVITY: CREATING YOUR FIRST CSS FILE
  1. You should still have the “Coding 101” folder open in your editor
  2. Create a new file from the editor, save the file as “style.css”
  3. Add a few CSS properties(color, background-color, font-family, text-align) to the content created in the last activity
  4. Connect your external CSS file to your HTML file
  5. Open up the HTML file in Google Chrome
